I Am Learning

aaronwatts@dev 23 May 2026

I've been learning again. Specifically React recently - I did this whole thing once before during the switch from class based to function based components, at a time when the documentation wasn't really keeping up with all the changes that were being made to React. That period appears to be over (for now), and I'm learning React just to get a better feel for Front End Frameworks now that native Web Components appear to be becoming widely available.

Whether or not I actually use what I learn as yet to be seen. But if there are native technologies, I prefer to understand them where I can. It might seem a little odd learning React to learn Web Components, and I might be wasting precious time. But I see it as a Front End Framework on rails. I bit like using JQuery as a way to get to grips with the DOM faster than with vanilla JavaScript (that is not how I learned DOM, but an argument could be made for it). React has also become quite ubiquitous and as a result there's a heap of learning resources for it.

I've also been quite busy with my Coolify instance. It's given me the opportunity to set up a WordPress site from scratch. I've only previously edited WordPress website content, and having gone through the set up process, it is astonishingly easy. I can appreciate why it's the best supported and most widely available way to run a full stack website. I might even have a project in mind for it. I'm hoping my tendency for minimalism will keep me out of the infamous plugin hell.

From the collected hours I've spent learning recently, and also not learning, it's become clear to me how much less time I have lately than when I was last here. I'm not entirely sure what's changed. I think it might be my day job getting in the way. But I guess that's just life, and I enjoy my job enough to just put up with it for now. I also need to be able to fund my hobbies and projects, so tit for tat.
